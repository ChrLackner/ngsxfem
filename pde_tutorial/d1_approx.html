1 Small inspection gimmicks:

1.1 Next, we want to do some simple inspection of the finite element space.

To this end, we print the degrees of freedoms corresponding to each element. This functionality is provided by PrintDofs: 

16: def PrintDofs(pde,mesh,fes):
17:     print ("Printing dofs per element:\n\n")
18:     for i in mesh.Elements():
19:         print("dofnrs of element", i, ":\n", fes.GetDofNrs(i))

1.2 To visualize the test functions (especially the XFEM ones) we use the following function:

20: def ShapeTest(pde,u):
21:     print ("Shape test:\n")
22:     u[:][:] = 0
23:     for i in range(u.size):
24:         print ("i = ", i ,".")
25:         u[:][i-1] = 0.0
26:         u[:][i] = 1.0
27:         Redraw(blocking=True)
28:         sleep(1)
